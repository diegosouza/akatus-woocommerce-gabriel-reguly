== Description ==

### Adds Brazilian payment gateway Akatus to WooCommerce.


The WooCommerce Akatus Gateway plugin enables integration with WooCommerce and the Akatus Payment Gateway (for Brazilian Merchants). 

Visit https://www.akatus.com for more details on using Akatus.

Please notice that WooCommerce must be installed and active.



== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
1. Activate the plugin
